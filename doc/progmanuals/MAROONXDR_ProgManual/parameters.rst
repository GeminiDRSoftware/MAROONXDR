**********
Parameters
**********

This section documents all the parameter configuration classes available in the MAROONX Data Reduction pipeline.
These parameters control the behavior of primitives and are organized by processing stage and functionality.

.. note::

   Parameter classes define the configuration options for each primitive.
   Each parameter includes type information, default values, and validation rules.
   Parameters can be modified through configuration files or programmatically.
